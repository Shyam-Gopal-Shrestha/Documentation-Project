Operators
In JavaScript, an operator is a special symbol or keyword that performs an operation on one or more operands (variables or values) and produces a result. JavaScript supports various types of operators, including arithmetic, assignment, comparison, logical, bitwise, and more.

Variables
In JavaScript, variables are used to store and manipulate data. They are containers that hold values that can be referenced and manipulated throughout a script. Variables can hold various types of data, including numbers, strings, arrays, objects, functions, and more. The type of data a variable can hold is not explicitly declared in JavaScript; it is dynamically determined at runtime.
Defining Variables
Variables can be mainly defined in three ways in JS.
1. Using "Var" Keyword: Historically used to declare variables. Variables declared with var are function-scoped or globally scoped, but not block-scoped.
2. Using "Let" Keyword: Introduced in ES6, let allows block-scoping. Variables declared with let can be reassigned.
3. Using "Const" Keyword: Also introduced in ES6, const declares variables that cannot be reassigned. They are also block-scoped.

Variable names in JavaScript can include letters, digits, underscores, and dollar signs. They must begin with a letter, an underscore (_), or a dollar sign ($). Variable names are case-sensitive.

Variables example:

var message = "Hello, World!";
let age = 30; // Variable declaration and initialization
let firstName; // Variable declaration without initialization
firstName = "John"; // Variable assignment
const PI = 3.14; // Constant declaration (cannot be reassigned)
Data Types
A data type is a classification that specifies which type of value a variable can hold in a programming language or system. It defines the operations that can be done on the data, the meaning of the data, and the way values of that type can be stored. JavaScript has several built-in data types:
1. Primitive Data Types:
Number: Represents numeric values. It includes integers and floating-point numbers.
let num = 10;

String: Represents textual data, enclosed in single or double quotes.
let str = "Hello, World!";

Boolean: Represents a logical value indicating true or false.
let isTrue = true;

Undefined: Represents a variable that has been declared but not assigned a value.
let undef;

Null: Represents an intentional absence of any value.
let n = null;

Symbol: Introduced in ES6, represents unique identifiers. Symbols are primarily used as property keys in objects.
let sym = Symbol("unique");

2. Non-primitive (Reference) Data Types:
Object: Represents a collection of key-value pairs, where values can be of any data type, including other objects.
let obj = { key: "value" };

Array: Represents an ordered list of values. Arrays can contain values of any data type, including other arrays and objects.
let arr = [1, 2, 3];

Function: Represents a reusable block of code that performs a specific task. Functions are also objects in JavaScript.
function greet() {
return "Hello!";
}

JavaScript also provides special values like NaN (Not-a-Number) and Infinity to represent certain mathematical situations.
